Practise tracks

Practise tracks contain recordings of carols with the specified part played on top. Most tracks are set up so that the original recording should come out of one speaker/headphone and the harmony track from the other.

You should be able to play these tracks on any computer, tablet or phone that supports MP3 playback (most modern devices do). If you need a copy of these tracks on CD then please ask.
